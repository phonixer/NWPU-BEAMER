# NWPUBeamer LaTex ppt npu 西北工业大学ppt模板
西北工业大学主题的Beamer模版  ppt

基于清华大学和东南大学的BEAMER-ppt模板创建了该模板
为了方便大家学习，清华大学beamer的东西并没有删除，相关代码只是屏蔽，打开可以使用
